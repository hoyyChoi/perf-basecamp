**측정 → 분석 → 개선**

## 작업 현황

### 측정

- 성능 점수 (lightHouse)
- 크기

  - 히어로 이미지 크기
- 속도 (프랑스 파리 기준)
  - 1번째 lcp 로드
  - 2번째 lcp 로드
- 프레임
  - Dropped Frame
  - Partially Presented Frame
